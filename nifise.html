The Story

Started from actual zero

I'm Victor — "Nifise" online. I started documenting my trading journey publicly with nothing but the willingness to show the real process, mistakes included. No borrowed screenshots, no "I made it" packaging.

My approach is built on HORC and MSNR — a multi-timeframe, top-down system (monthly → weekly → daily → 4H) that only takes entries when a setup aligns across every level. It's not about calling every move right; it's about only acting when the odds are actually stacked in my favor.

I'm still early. I post the losses as openly as the wins, because that's the only version of this that's actually useful to anyone else starting from the same place.
